Keyboard Shortcuts

Tab

normal navigation.

Space

open and close folders.

Enter

open and close folder or open a link.

Ctrl + q

close any kind of modal.

Ctrl + c

copy folder title or link url on clipboard

d

focus the parent folder

Ctrl + Enter

on links, opens the link in a contrary way that the open behavior

Ctrl + mouse click

on links, opens the link in a contrary way that the open behavior

Shift + Enter

on links, opens the link in a new windows

Shift + mouse click

on links, opens the link in a new windows

f

create new folder inside the current folder.

Shift + f

create new folder inside the parent folder.

b

create new bookmark inside the current folder.

Shift + b

create new bookmark inside the parent folder.

v

opens the "create bookmark modal" with empty fields to create a new bookmark inside the current folder.

Shift + v

opens the "create bookmark modal" with empty fields to create a new bookmark inside the parent folder.

r

remove link or folder.

e

edit folder or link.

u

undo last remove link or folder (in the undo timeout).

s

sort the current folder.

Shift + s

sort the parent folder.

o

on foldres, open all links in the current folder.

Shift + o

on folders, open all links in the current folder in a new window.

Shift + i

on folders, open all links in the current folder in a incognito window.

m

open or close the "more () modal".

k

open or close the "keyboard () modal".
